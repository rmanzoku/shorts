# Conte Creation Workflow

調査メモからコンテ（動画台本）を作成する: $ARGUMENTS

> 全体フロー: ネタ提供 → `/research` → 調査レビュー → **`/conte`** → コンテレビュー → `/generate` → `/improve` → アップロード

## Step 1: 入力確認

調査メモ（`.research.md`）を読み込む:

```bash
cat <input_path>   # contes/NNN_slug.research.md
```

調査メモがない場合は、ユーザーが直接提供したテキスト・メモ・URLから作成する。
その場合はファクトチェックを特に慎重に行う。

## Step 2: 構成設計

調査メモの「動画の切り口候補」から最適な切り口を選び、構成を決める:

- **フック**: 冒頭3秒で視聴者を引き込む事実・疑問・数字を選ぶ
- **本体**: 3〜6シーンで「なぜ？」に答える構成
- **締め**: コメントを促す問いかけで終わる

## Step 3: コンテ作成

`contes/NNN_slug.md` に出力する（`.research.md` と同じ連番を使う）:

```bash
ls contes/   # 連番確認
# 例: 002_topic.research.md があるなら → 002_topic.md
```

以下のフォーマットで `.md` ファイルを生成する:

```markdown
# <タイトル（興味を引く疑問形・数字入り推奨）>

## シーン 1
**映像**: <映像の説明（AI画像生成の指示になる）>
**ナレーション**: <TTS読み上げテキスト>
**数字**: <画面に大きく表示する数字・キーワード>

## シーン 2
**映像**: <映像の説明>
**ナレーション**: <TTS読み上げテキスト>
**数字**: <画面に大きく表示する数字・キーワード>

...（3〜6シーン）

## 説明
<1〜2文の要約（検索キーワードを含める）>

📌 目次
0:00 <トピック1>
0:XX <トピック2>
...

🤖 この動画は映像・音声・字幕すべてAIで生成しています

#政治 #時事問題 #解説 #ニュース #日本 #ショート #政治解説 #AI解説 #AI生成 #<トピック固有タグ1> #<トピック固有タグ2>
```

### 台本ガイドライン

- です・ます調で統一
- 読点は1文に2つ以内
- ナレーション合計: 60〜90秒（約350〜525文字）
- シーン数: 3〜6
- 偏向・扇動的表現の禁止
- 映像指示は具体的に（「街の風景」ではなく「渋谷スクランブル交差点の夜景、ネオンが反射する雨上がりの路面」）

### 映像指示の注意点

- テキスト・文字・ロゴを含む映像指示は避ける（AI画像で文字化けするため）
- 人物の顔アップは避ける（AI画像の弱点）
- 風景・建物・物体・抽象的なイメージを推奨

### アルゴリズム最適化

- **冒頭フック（シーン1の最初の1文）**: 最初の12文字以内に数字または疑問符を含めること
  - 良い例: 「9ヶ月で11議席。この新党を知っていますか？」（数字→問いかけ）
  - 良い例: 「あなたの社会保険料、実は年間○○万円です」（個人に関係する数字）
  - 悪い例: 「今日は○○について解説します」（弱い・文脈説明）
  - 悪い例: 「2026年2月の衆院選で、ある新党が…」（年号から始まる文脈説明）
- **末尾の問いかけ（最終シーンの最後の1文）**: 「あなたは」を含む直接的な問いかけで終わる
  - 良い例: 「あなたはこの政策に賛成ですか？コメントで教えてください」
  - 良い例: 「○○と△△、どちらが正しいと思いますか？」
  - 悪い例: 「今後の展開に注目です」（受動的・コメントを促さない）
- **数字オーバーレイ**: 全シーンに `**数字**:` フィールドを含めること（画面に大きく表示される）
  - 例: `**数字**: 381万票`、`**数字**: 平均年齢39歳`
- **情報密度**: 比較・ランキング・意外な数字など「保存したくなる」要素を1つ以上含める
- **動画尺の確認**: ナレーション合計が60秒以上になるよう文字数を調整（約350文字以上）

### 説明セクション

ハッシュタグルール:
- プロファイルの固定タグ（9個）+ トピック固有タグ（1〜3個）= 合計10〜12個
- トレンドに関連するタグがあれば追加

## Step 4: セルフチェック

作成したコンテを以下の観点で確認する:

- [ ] タイトルは疑問形または数字入りで興味を引くか
- [ ] シーン1の冒頭12文字以内に数字または疑問符が含まれているか
- [ ] 最終シーン末尾に「あなたは」を含む直接的な問いかけがあるか
- [ ] 全シーンに `**数字**:` フィールドがあるか
- [ ] 比較・ランキング・意外な数字など「保存したくなる」要素が1つ以上あるか
- [ ] ナレーション合計は60〜90秒（350〜525文字）の範囲か
- [ ] 映像指示にテキスト・顔のアップが含まれていないか
- [ ] `## 説明` セクションにハッシュタグ・目次・AIラベルがあるか
- [ ] 偏向・扇動的表現がないか

## Step 5: ユーザーにレビュー依頼

作成したコンテをユーザーに提示し、以下を確認してもらう:

- 事実関係に誤りがないか
- 構成・シーン分割は適切か
- 映像のイメージは意図通りか
- ナレーションのトーンは適切か

修正が必要な場合はコンテを更新する。
レビュー完了後、`/generate contes/NNN_slug.md` で動画生成に進む。
出力先は `output/NNN_slug.mp4` を推奨。
